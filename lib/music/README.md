# 音频组件

## 项目简介

用于专题播放音乐使用的组件。

## 相关人员

- 前端开发： [王进](mailto://wangjin@demo.cn)

### 基本使用

```javascript
const Music = require('fe-core/lib/music');
Music.inject(assetsURL('music/bgm.mp3'));
```
